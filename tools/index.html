Pro Tip for Realtors

When listing a home or preparing for a sale, pull both the property appraiser record (for assessed value, square footage, and exemptions) and the permit history (to verify that additions, renovations, and roof replacements were properly permitted). Unpermitted work can delay closings and cause insurance complications. Some cities run their own permit systems separate from the county — if you don't find a permit in the county portal, check the city-specific links.
